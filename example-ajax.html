Cynteka Pivot Table Ajax Example

In this example Cynteka Pivot plugin loads data from Google Spreadsheet data source. You can specify you own spreadsheet URL. Please create your own spreadsheet, add column title in the first row (start from column A) and use the last column as value column that should be used inside pivot table (population, costs, area etc). Publish the document in the Internet, after that it will be available for pivotting on this page. Each expand event generates separate request to the datasource. Pivot table feeds only necessary portion of data. It allows to get rid of loading huge portion of data to the browser.

Predefined data sources

Restaurants, Goods, Stores, Money... Cities, Rivers, Countries, Population...

You can select your own datasource
